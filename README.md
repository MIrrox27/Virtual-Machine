# Virtual-Machine
Simple Virtual Machine for doing user's commands.

### Commands
At this moment, program can input 11 codes:
  * ERR <code> - exit with error code
  * HALT - exit from program
  * PUSH <element> - add element to stack
  * ADD 
  * SUB 
  * MUL 
  * DIV 
  * MOD  
  * POW  
  * SQRT 
  * JUMP <ip> - jump to new ip
  * JIF <ip> - jump to new ip, if ended element not 0 - true
  * PRINT - print ended element to console
