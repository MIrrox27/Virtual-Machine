# Virtual-Machine
Simple Virtual Machine for doing user's commands.

### Commands
At this moment, program can input 11 codes:
  * HALT = 0 - exit from program
  * PUSH = 1 - add element to stack
  * ADD = 2
  * SUB = 3
  * MUL = 4
  * DIV = 5
  * MOD = 6 
  * POW = 7 
  * SQRT = 8 
  * JUMP = 9 - jump to new ip
  * JIF = 10 - jump to new ip, if ended element not 0 - true
  * PRINT = 11 - print ended element to console
